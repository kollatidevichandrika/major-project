OVERVIEW:
This project analyzes student spending behavior and lifestyle patterns using survey data. The goal is to understand how financial habits such as spending, savings, and eating-out frequency influence overall expenditure.

OBJECTIVES:
Analyze relationships between spending, savings, and lifestyle habits
Perform Exploratory Data Analysis (EDA)
Identify key factors affecting student spending
Build a predictive model for estimating spending

FEATURES:
Data cleaning
EDA
Dashboard (Plotly)
Linear Regression model

TOOLS:
Python, Pandas, Plotly, Scikit-learn

HOW TO RUN:
Install requirements
Run notebooks
Run dashboard_code.py
